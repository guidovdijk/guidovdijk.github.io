Aps Milliue

APS-Milieu B.V. hun website was erg verouderd. Ze vonden dat als ze niet meer de moderne kant op gingen met hun uistraling dat ze klanten verliezen.
Ik heb tijden mijn stage bij FBI Design de website mogen realiseren van begin tot eind met wordpress, PHP, jQuery en javascript

website

De klant wou graag dat de vierkantjes niet uit proportie gaan, maar echt vierkantjes blijven.
Ook stond in het ontwerp dat de vierkantjes voor de helft over een foto gaan. Hiervoor heb ik met jQuery een code geschreven dat van elk blok de "width" pakt en die value in de "height" zet.
Om de blokken over de helft van de afbeelding te zetten heb ik in de html alle blokken bij elkaar in een div gezet en op de div een negatieve margin gezet dat gelijk is aan de width van de blokken.
