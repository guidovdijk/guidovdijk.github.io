PokeApi

De PokeApi is een open source API waarmee ik heb geoefend met React.js. Ik heb de API-calls gemaakt met Axios en de webapp ge-host op Heroku App.
Voor de styling heb ik gebruik gemaakt van Sass.

De reden dat ik deze web app heb gemaakt is omdat ik meer wou leren over JavaScript, met name de front end ervan. Ik had gekeken welke libaries en talen er op dit moment een goede ligging en waarschijnlijk gaan groeien in de komende jaren. Na een tijdje zoeken kwam ik uit op React.js en besloot er verder mee te gaan.
